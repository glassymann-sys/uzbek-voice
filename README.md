# 🎙️ O'zbek TTS — Gemini AI

O'zbek matningizni **Google Gemini AI** yordamida tabiiy ovozga aylantiring.

## ✨ Xususiyatlar

- 🔊 Gemini 2.5 Flash / Pro TTS modellari
- 🎭 8 xil ovoz (ayol va erkak)
- 📝 5000 belgigacha matn qo'llab-quvvatlash
- ⬇️ WAV formatida yuklab olish
- 📱 Mobil qurilmalarga moslashgan dizayn
- 🔒 Brauzerda ishlaydi — ma'lumotlar saqlanmaydi

## 🚀 Ishlatish

1. `index.html` faylini brauzerda oching
2. [Google AI Studio](https://aistudio.google.com/app/apikey) dan bepul API kalit oling
3. Kalitni kiriting, matn yozing → **Ovoz yaratish** tugmasini bosing
